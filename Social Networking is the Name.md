Social Networking is the Name of the application

SOCIAL NETWORKING MVP - A social networking site for professional African Americans 

Features Provided for the MVP

* User Profiles - Basic user registration and profile management with avatar, bio, and privacy settings.
* Friend/Follower System - Users can send/accept friend requests or follow others to build a network.
* Posting System - Users can post text updates, upload images, and share short videos.
* Content Feed - Home feed with recent posts from friends or followed users.
* Direct Messaging - One-on-one private messaging between connected users.
* Responsive UI - Mobile-friendly interface with bright blue theme and accessible layout.
* Authentication - Secure sign-up/sign-in via email using AWS Cognito.
* Media Upload - Image/video content stored and served using Amazon S3 and CloudFront
* geared towards Professional African Americans



Tech stack: 

* next.js
* tailwind.css
* typescript
* shadcn/ui

* AWS Lambda - Serverless compute for handling backend logic(or suggest a better solution)
* Amazon CloudFront - CDN for delivering frontend and media content globally (or suggest a better solution)
* Amazon Cognito - User authentication and identity management (or suggest a better solution)
* CloudWatch - Monitoring, logging, and alerts for all services (or suggest a better solution)
* DynamoDB - NoSQL database for storing user, post, and message data (or suggest a better solution)
